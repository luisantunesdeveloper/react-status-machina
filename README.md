# react-status-machina

> binds react to status-machina through react context api

## Install

```sh
npm i react-status-machina
```

## Development

open a pull request when you're ready.  
don't forget to add your tests.

```sh
git clone https://github.com/luisantunesdeveloper/react-status-machina.git
cd react-status-machina && npm i
```

### Tests

```sh
npm test
```

```sh
npm run test:watch
```
